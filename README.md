# Mario platform game
A platform game made with pygame. This project was made as part of the *Premier projet d'informatique* course given by Laurent Mathy at the University of Li√®ge during the academic year 2018-2019.

## Usage
Install python3, pygame and run :
```bash
$ python3 mario.py
```

## Keys in the game
- Move with the arrow-scroll keys.
- Jump with the space key.
- Respawn with the 'R' key.
